## Описание
Графическое приложение на Python (Tkinter) для генерации случайных цитат. 
Позволяет просматривать предопределённый набор цитат, добавлять свои, 
фильтровать историю по автору и теме, а также сохранять/загружать историю генераций в формате JSON.

## Возможности
- Случайная генерация цитаты из встроенной базы
- Добавление новых цитат с валидацией пустых полей
- Фильтрация истории по автору и теме
- Автосохранение истории в `quotes_history.json` при закрытии
- Ручная загрузка/сохранение через интерфейс
- Чистый интерфейс на `tkinter` + `ttk`
